## 프로젝트
### 개요
* 로스트아크 일일, 주간 컨텐츠 체크 및 공유 사이트 
* 일일 컨텐츠 : 카오스 던전, 가디언 토벌 
* 주간 컨텐츠 : 군단장 레이드, 어비스 던전 
* 로스트아크 오픈 API의 거래소, 경매장 등의 데이터를 호출해 예상 수익 계산
